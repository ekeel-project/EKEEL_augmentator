Content folder of all developed elements for the EKEEL Project  

**(dir)Blueprints**: Folder containing developed blueprint actors and components.  
**(dir)FX**: Folder containing all elements related to visual effects and particle effect systems (i.e. Niagara Systems).  
**(dir)Haptics**: Folder containing data assets related to haptics.  
**(dir)Images**: Folder containing image assets (i.e. button icons).  
**(dir)Inputs**: Folder containing elements related to input actions and control mappings.  
**(dir)Json**: Folder containing json files (i.e. graph_example.json).  
**(dir)Levels**: Folder containing unreal engine environments.  
**(dir)Logs**: Folder containing custom log files generated by the system.  
**(dir)Materials**: Folder containing materials for meshes and effect systems.  
**(dir)Meshes**: Folder containing custom meshes.  
**(dir)ModernLibrary**: Folder containing blueprints, meshes, texture assets for a library environment.  
**(dir)Movies**: Folder containing media files for use by the system.  
**(dir)Structures**: Folder containing data structures and enumerators for modules.  
**(dir)Widgets**: Folder containing blueprints and elements related to user widgets.  

**(dir)(auto)Collections**: Folder containing groups of assets related to specific characteristics using the unreal engine collection functionality.  
**(dir)(auto)Developers**: Folder containing custom data related to each developer.
